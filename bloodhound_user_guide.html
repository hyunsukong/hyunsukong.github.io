Python Dependencies

Bloodhound requires the following Python packages:

  • Astropy
  • h5py
  • NumPy
  • pandas
  • SciPy

Required Input Data

Bloodhound requires the following input datasets: *** make sure to write where these files need to be! ***

  • Simulation snapshots
  • Rockstar halo catalogs
  • Rockstar .bin files
  • Consistent-trees full merger tree data: tree.hdf5
  • Snapshot time information: snapshot_redshift_scale-factor.txt
